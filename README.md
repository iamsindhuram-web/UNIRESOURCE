# UniResource Web Application

## Description

UniResource is a full-stack web application developed to manage and access resources efficiently. The system provides a structured interface where users can interact with the application through a web browser, while the backend handles data processing and storage.

The application is designed using a client-server architecture. The frontend provides the user interface, while the backend manages business logic and communicates with the database. MongoDB is used to store and retrieve application data.

---

## Working of the Application

The user interacts with the frontend interface through a browser. When a request such as login is made, it is sent to the backend server built using Node.js and Express.js. The backend processes the request and communicates with MongoDB Atlas to validate or store data. The response is then sent back to the frontend and displayed to the user.

---

## Deployment

The backend server is deployed using Render, which allows the application to run on the internet. MongoDB Atlas is used as a cloud database service to store application data securely. The application is accessible through a public URL.
